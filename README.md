# Finetuned LLM Structured Company Analysis

This project explores LoRA finetuning of Qwen2-7B to generate structured company analysis from company descriptions.

## Project Components

- Synthetic dataset generation
- Data preprocessing
- LoRA finetuning
- Structured analysis generation

## Dataset

300 synthetic company descriptions were generated and converted into structured analysis format.

## Model

Qwen2-7B + LoRA finetuning
